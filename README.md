# U.S._dollar_exchange_rates
We analyzed a 34-year monthly series of U.S. dollar exchange rates with 408 observations arranged as a 34×12 year-by-month matrix [1]. We imported the series from a plain text file and reshaped it into a 34×12 matrix in R.








[1] Husain, M. Adam, M. Shitan, and A. Fitrianto, "Exploratory Extreme Data Analysis for Farmer Mac Data," Malaysian Journal of Mathematical Sciences, vol. 11, pp. 1-16, 2017.
